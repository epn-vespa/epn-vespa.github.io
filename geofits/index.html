GeoFITS: FITS format for Planetary Surfaces.

Planetary Sciences are evolving rapidly from visual inspection and by-hand analysis to a more automated quantitative analysis and only final product visualization. One of the VESPA goal is to extend FITS and WCS metadata description to Planetary Sciences, and to insert FITS in the GIS interoperability schema.

VESPA FITS projects

  • GeoFITS convention proposal
  • FITS World Coordinate System translation in GDAL Virtual Header
  • GDAL complete FITS implementation
